How to Use Datta Fish:

Catches

Datta Fish allows users to easily input, store and view catches. In order to access these features, simply click on the "Catches" button at the top of any page or click on this link.

Here you can see the button "Create Catch" and a listing of all your logged catches.

If you want to create a catch, click the button labled "Create Catch", enter all information, and click submit. You will then be shown a detailed view of your catch.

On "Catches" main page, you can access a detailed view of any of your catches by clicking on one. The detailed view will show you all information you entered about that catch and allows you to edit and delete the catch.


Data Track

Datta Fish also allows users to view all logged catches as an organized and sorted listing. This allows you to view trends in your catches such as the highest weight by location or the best lure for a species of fish. In order to access these features, simply click on the "Data Track" button at the top of any page or click on this link.

In this example on the left, we start at the Data Track page. We then set the first query to "Species", the second query to "Location" and the name of the second query is set to "Blue Pond". When clicking the submit button, the species caught in Blue Pond are shown from most common to least common. Under each species is the logged catches that meet both of the entered criteria. You can click on any of these catches to be show a detailed view of that catch. If you want enter different data track queries, you can click on the "New Query" button.

Tip: Setting 2 queries to the same thing, such as "Species", and entering the name of that query, such as "Largemouth Bass", will return only catches with that specific criteria, in this case all Largemouth Bass catches.
